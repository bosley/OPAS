# Response 

A response from the NIS will be as follows: 

```json
{
  "nss": false,
  "nrs": true,
  "nos": true,
  "nps": false
}
```

### Members

***nss*** - _Node Sender Service_
***nrs*** - _Node Receiver Service_
***nos*** - _Node Oracle Service_
***nps*** - _Node Pillar Service_

### Values

[***true*** => Service Active, ***false*** => Service Not Active]
