# Command Line Interface to Node

## Test
test
    net
        - Tests networking functions (all)


> Example : test net

## Generate 
generate
    address
        - Generate a new address

    tx
        - Bring up transaction generation menu

> Example : generate tx 

## display

    screen
        - Show display screen of node activity

    peernum
        - Shows number of peers
    


> Example : display peernum
